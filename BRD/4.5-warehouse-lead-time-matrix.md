# 4.5. Warehouse Lead Time Matrix

| Permission | Description |
|---|---|
| **Can view Warehouse Lead Times** | View warehouse lead time configurations by order type, group, and quantity break |
| **Can manage Adjustments and Overrides** | Define and update adjustments and overrides at order type or group level |
| **Can Add Personalisation Rules** | Can add personalisation lead time rules per order type |
| **Can Edit Personalisation Rules** | Can edit personalisation lead time rules per order type |
| **Can delete Personalisation Rules** | Can delete personalisation lead time rules per order type |
| **Can Add Groups** | Can add a new group |
| **Can Edit Groups** | Can edit groups |
| **Can Delete Groups** | Can delete groups |
| **Can Add Quantity Breaks** | Can add a new quantity break |
| **Can Edit Quantity Breaks** | Can edit quantity breaks |
| **Can Delete Quantity Breaks** | Can delete quantity breaks |
