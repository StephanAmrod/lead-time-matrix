# 4.2. Special Dates Manager

| Permission | Description |
|---|---|
| **Can view Special Dates** | View all configured special dates, including public holidays and shutdown periods |
| **Can Add Special Dates** | Add a new special date |
| **Can Edit Special Dates** | Edit special dates |
| **Can Delete Special Dates** | Delete a special date |
| **Can Add Special Dates Categories** | Add a special date category |
| **Can Edit Special Dates Categories** | Edit special date categories |
| **Can Delete Special Dates Categories** | Delete a special date category |
