# 4.3. Branding Departments Manager

| Permission | Description |
|---|---|
| **Can view Branding Departments** | View branding departments, print codes, invoice codes, and setup codes in a hierarchical structure |
| **Can Add Invoice Codes** | Can add invoice codes linked to print codes (access to this to be manageable) |
| **Can Edit Invoice Codes** | Can edit invoice codes linked to print codes |
| **Can Delete Invoice Codes** | Can delete invoice codes linked to print codes |
| **Can Add Setup Codes** | Can add setup codes linked to print codes |
| **Can Edit Setup Codes** | Can edit setup codes linked to print codes |
| **Can Delete Setup Codes** | Can delete setup codes linked to print codes |
| **Can Add Additional Charges** | Can add additional charge codes per branding department |
| **Can Edit Additional Charges** | Can edit additional charge codes per branding department |
| **Can Delete Additional Charges** | Can delete additional charge codes per branding department |

### Note

Currently, once a branding code is in this matrix and is invoiced, it creates a job card in the Amtrack engine. Now that it is being developed into Moyo, we will have to upgrade Amtrack to read from here.
