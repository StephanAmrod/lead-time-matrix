# 4.4. Production Lead Time Matrix

| Permission | Description |
|---|---|
| **Can view Production Lead Times** | View production lead time matrix across departments, groups, and quantity breaks |
| **Can manage Adjustments and Overrides** | Define and update adjustments and overrides at branding department or group level |
| **Can Add Personalisation Rules** | Can add personalisation lead time rules per branding department |
| **Can Edit Personalisation Rules** | Can edit personalisation lead time rules per branding department |
| **Can delete Personalisation Rules** | Can delete personalisation lead time rules per branding department |
| **Can Add Groups** | Can add a new branding group |
| **Can Edit Groups** | Can edit branding groups |
| **Can Delete Groups** | Can delete branding groups |
| **Can Add Quantity Breaks** | Can add a new quantity break |
| **Can Edit Quantity Breaks** | Can edit quantity breaks |
| **Can Delete Quantity Breaks** | Can delete quantity breaks |
